# Quickstart: Phase I Todo App

## Prerequisites
- Python 3.13+
- uv (recommended)

## Setup
```bash
# Initialize project with uv
uv init
```

## Running the Application
```bash
python src/main.py
```

## CLI Interaction
1. **Add Task**: Enter description when prompted.
2. **View Tasks**: Displays ID, Description, and Status.
3. **Update Task**: Provide ID and new description.
4. **Delete Task**: Provide ID to remove.
5. **Mark Complete**: Provide ID to toggle status.
6. **Exit**: Shuts down the application.
